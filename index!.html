ClearRead

A text simplifier prototype for accessible learning

About this project

ClearRead rewrites dense academic text into shorter sentences with simpler words, and adds a concrete real-world example for a small set of recognised topics, designed to reduce reading load for students with dyslexia, ADHD, or anyone who benefits from plainer language. This is a working front-end prototype: it uses a rule-based dictionary of word swaps and topic-matched examples, so it runs instantly with no API key. A production version would connect to an AI model such as Claude or GPT, which could simplify any topic and generate a relevant example on the fly, rather than relying on a fixed list of recognised topics like this prototype does.
