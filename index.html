ClearRead

A text simplifier prototype for accessible learning

About this project

ClearRead rewrites dense academic text into short, plain-English sentences, designed to reduce reading load for students with dyslexia, ADHD, or anyone who benefits from simpler sentence structure. This is a working front-end prototype: it currently uses a rule-based simplifier (splitting long sentences logically) to demonstrate the concept without requiring an API key. A production version would connect to an AI model such as Claude or GPT for true semantic simplification at scale, while keeping this same interface.
